# 🎯 Streak Flow

Track your daily habits, visualize your streaks, and crush your goals. A sleek and modern habit tracker built using React and Tailwind CSS.

## ✨ Features

- **Daily Habit Tracking** - Mark habits as complete each day with a single click
- **Streak Visualization** - See your progress with beautiful streak counters
- **Goal Management** - Set and monitor personal goals
- **Modern UI** - Sleek, intuitive interface built with React and Tailwind CSS
- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile
- **Progress Analytics** - Track your consistency over time

## 🚀 Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/nadrabibi92-cloud/streak-flow.git
cd streak-flow
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

The application will open at `http://localhost:3000`

## 📖 Usage

1. **Add a Habit** - Click the "Add Habit" button and enter habit details
2. **Log Daily Progress** - Check off completed habits each day
3. **View Streaks** - Watch your streak count grow as you maintain consistency
